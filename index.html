How to Use

使い方

洗顔後の清潔な肌に、1〜3滴を手のひらで温めてからお使いください。

Use 01 / Face
FACE IMAGE

顔のスキンケアに

洗顔後、または化粧水の後に1〜2滴。手のひらに広げて、顔全体を包み込むようになじませます。ツヤ感のある肌に。

Use 02 / Hair
HAIR IMAGE

髪のケアに

タオルドライ後の濡れた髪に1〜2滴。毛先を中心になじませてからドライヤーで乾かします。乾いた髪のまとまりにも。

Use 03 / Body
BODY IMAGE

からだの保湿に

入浴後の濡れた肌、またはボディクリームの前に。全身に薄くなじませるだけで、乾燥が気になる部分もしっとりと整います。

FAQ

よくあるご質問

植物由来の成分を中心に使用した低刺激処方です。ただし、肌質や状態により合わない場合もございます。初めてお使いになる際は、腕の内側などで少量パッチテストを行ってからご使用ください。
朝・夜どちらにもお使いいただけます。夜は洗顔後の保湿ケアとして。朝は化粧下地前の保湿として、または髪のスタイリングにもご活用いただけます。ライフスタイルに合わせてお使いください。
顔のみに1〜2滴/日ご使用の場合、目安として約1〜2ヶ月程度です。顔・髪・からだに使う場合はご使用量によって異なります。
PROTEAは少量生産・出来立て発送にこだわっているため、在庫状況により発送までお時間をいただく場合があります。最新の在庫状況・発送スケジュールは公式LINEおよびSTORES販売ページでご確認ください。
商品の品質には万全を期しておりますが、万が一不良品や誤送があった場合は、到着後7日以内にSTORESのメッセージ機能またはSNSのDMよりご連絡ください。お客様都合によるご返品・交換は対応しかねます。あらかじめご了承ください。
グレープフルーツの香りは、明るくすっきりした爽やかな香りで朝のケアにもぴったりです。今回新しく登場した香りです。ベルガモットの香りは、落ち着きのある上品な柑橘系の香りで、夜のリラックスタイムにもよく合います。どちらも同じ処方・品質ですので、お好みや使うシーンでお選びください。
